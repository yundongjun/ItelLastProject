# ItelLastProject

# 처음 시작

1. git 다운로드
2. git init으로 git repository 생성
   ```
   git init
   ```
3.  원격 repository 가져오기
  ```
   git clone https://github.com/yundongjun/ItelLastProject.git
  ```
# 폴더형식

1. stm32폴더에는 stm32관련된 내용 (성준)
2. cctv폴더에는 cctv관련된 내용 (박진수)
3. ros폴더에는 ros, opencr 관려된 내용 (윤동준, 설영현)
